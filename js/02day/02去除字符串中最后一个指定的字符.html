去除字符串中最后一个指定的字符

思路：把指定字符串反转，replace() 把目标字符替换成空字符串， 再把新生成的字符串反转回来 

    
        function delLastChar1(str, target) {
            if (typeof str === 'string') {
              return str
                .split('')
                .reverse()
                .join('')
                .replace(target, '')
                .split('')
                .reverse()
                .join('');
            } else {
              alert('请确认str类型');
              return false;
            }
        }

思路：用正则表达式匹配到字符串中要删除的字符，replace直接替换掉

        
        function delLastChar2(str, target) {
            let reg = new RegExp(`${target}(?=([^${target}]*)$)`);
            return str.replace(reg, '');
        }
